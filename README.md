# Fake News Detection using Neural Networks

## Introduction
This project aims to classify news articles as real or fake using machine learning techniques.

## Methodology
- Data preprocessing (cleaning, stopword removal)
- Feature extraction using TF-IDF
- Model training using Neural Networks / BERT
- Evaluation based on accuracy

## Technologies Used
- Python
- Scikit-learn
- TensorFlow / PyTorch
- NLP techniques

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run preprocessing:
   python src/preprocessing.py

3. Train model:
   python src/model.py

4. Predict:
   python src/predict.py
